# Trigger examples — documenting-design-system

## Should trigger

- "Aquí está el manual de marca en PDF, saca el design system"
- "Tengo el PPT de plantillas y capturas de la web, documenta la identidad visual"
- "Crea una skill de diseño para este cliente a partir de estos assets"
- "Extrae colores, tipografías y componentes del brand book"

## Should not trigger

- "Resume la llamada de discovery" → summarizing-discovery-calls
- "Escribe el tono de voz verbal" → tone-of-voice only, no visual extraction
- "Implementa el CSS en React" → engineering task after design-system exists
