# REPLACE Plugin Name (Cursor)

Usually generated from a Claude-first plugin using [../../adapters/to-cursor.md](../../adapters/to-cursor.md).

Default install path for local testing: `~/.cursor/plugins/local/<plugin-name>/`.

See [../../docs/best-practices/11-packaging-cursor-plugin.md](../../docs/best-practices/11-packaging-cursor-plugin.md).
