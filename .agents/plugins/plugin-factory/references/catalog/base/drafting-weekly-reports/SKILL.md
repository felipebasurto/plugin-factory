# Drafting Weekly Reports

## Goal

Turn raw data and bullets into a consistent management report draft with explicit data sources.

## When to use

- User provides Excel exports, KPI bullets, or ERP extracts.
- Recurring ops/commercial reporting.

## Do not use this skill when

- No data provided for the period.
- User needs live BI connection (mark phase 2).

## Inputs

Required: period and data (table, CSV, bullets).

Optional: audience (leadership, ops).

## Workflow

1. Confirm reporting period and metrics.
2. Structure: highlights, KPIs, issues, next week.
3. Narrate trends; cite source rows/files.
4. Flag missing or inconsistent data.

## Rules

- Do not invent numbers.
- Mark estimates clearly.

## Output

1. Report draft.
2. Data sources.
3. Data gaps.
