# REPLACE Plugin Name (Codex)

Usually generated from a Claude-first plugin using [../../adapters/to-codex.md](../../adapters/to-codex.md).

Add marketplace entry at repo `.agents/plugins/marketplace.json` or user `~/.agents/plugins/marketplace.json`.

See [../../docs/best-practices/10-packaging-codex-plugin.md](../../docs/best-practices/10-packaging-codex-plugin.md).
